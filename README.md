# CraftSphere

A modern Full-Stack Blog Application built with Next.js that allows users to sign in using Google Authentication and create, manage, and publish blog posts.

## Features

- Google Authentication (NextAuth)
- Secure Login & Logout
- User Dashboard
- Create Blog Posts
- Edit & Delete Blogs
- MongoDB Database Integration
- Responsive UI
- Protected Routes

## Tech Stack

- Next.js
- React.js
- NextAuth.js
- MongoDB
- Mongoose
- Tailwind CSS
- module.CSS
- JavaScript

## Project Structure

```
src/

```

## Installation

```bash
git clone https://github.com/iamsabiyakhan/CraftSphere.git

cd CraftSphere

yarn install

yarn dev
```
```
